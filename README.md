# Academic Collaboration App

**Project Status: Beta - Feature Development**

Empowering students and researchers to collaborate seamlessly on academic projects.

## Features
- Real-time collaborative editing
- Cloud Sync (Currently in Development)
- Social Auth Integration (Currently in Development)
- Integrated research tools
- Resource sharing repository
- Peer review system

## Setup
1. Clone the repo
2. Run 'flutter pub get'
3. Configure Supabase environment
4. Run 'flutter run'
